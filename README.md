# SkyCast 🌤️
> Your world. Your weather. Beautifully told.

## Live App
https://custom-weather-search.netlify.app/

## What's New in v2.0
- Wide location search (villages, towns, cities, states, countries)
- 10 distinct animated weather conditions
- Glassmorphism UI with smooth transitions
- 12-hour weather timeline
- Extended weather data (UV, pressure, visibility, sunrise/sunset)
- Day/Night mode detection
- Made by Sriram Davuluri

## Weather Conditions Supported
- ☀️ Clear Day
- 🌙 Clear Night
- ⛅ Partly Cloudy
- ☁️ Cloudy
- 🌫️ Foggy
- 🌦️ Drizzle
- 🌧️ Rainy
- ❄️ Snowy
- ⛈️ Thunderstorm
- 💨 Windy

## Tech Stack
Angular 19, Open-Meteo API, CSS Keyframe Animations,
Glassmorphism, Netlify

## How to Run
npm install
ng serve

## API
Open-Meteo — free, no key required
